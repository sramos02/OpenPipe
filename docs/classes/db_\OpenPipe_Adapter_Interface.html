An interface defining an adapter which bridges php based applications with OpenPipe. OpenPipe will call the adapter to load layouts and pipelets.

In essence the adapter is responsible for making sure that the php based application is instantiated, bootstrapped, and run appropriately to obtain the request element (either layout or pipelet)
