Implementation of an OpenPipe output interface that sends data as a standard HTML document Content pieces are used to construct a complete HTML document, placing CSS and JavaScript in proper placement, and inject each content piece within a pipelet place holder on the server side. It's important to note that no javascript is required to complete output on the client web browser while utilizing this output implementation
